// 01 — qué_es

Claude Code no te dice
cuánto consumes.
RETRO_DASH sí.

Trabajas en la terminal y los tokens vuelan a ciegas. RETRO_DASH lee los archivos de sesión locales (~/.claude/projects) y sincroniza el uso real de tu plan para enseñarte, sin abrir nada, lo que de verdad estás gastando.

01

Datos reales, no estimaciones

Lee cada sesión incremental: input, output y cache tokens, coste en euros por modelo, sesiones del día. Lo que ves es lo que hay.

02

Tus límites del plan, sincronizados

Ventana de 5 horas y límites semanales (todos los modelos: Sonnet, Opus, Haiku) sincronizados con tu cuenta. Coincide con el panel al digitar.

03

Flota sobre todo lo demás

Atajo global Alt+Shift+C para abrir/cerrar. Diseño CRT, ventana sin marco, siempre encima. Arrastrable, redimensionable, opaca o fantasma.

// 03 — cero_fricción

Tres pasos.
Cero configuración.

1

Descárgalo

ZIP portable para Windows. Binarios nativos para macOS y Linux sin entorno local de Claude Code.

2

Pulsa el atajo

Alt+Shift+C para abrir/cerrar el monitor flotante. Sin ventanas extras, sin clicks.

3

A vivir

Lo dejas siempre encima o lo escondes. La ventana vive, los datos cambian, la forma se queda.

Requiere tener Claude Code instalado y una sesión iniciada. No abre nada: solo lee los archivos locales en ~/.claude/projects bajo tu perfil. Solo corre en tu máquina.
